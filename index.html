Press A Key to Start

Instructions

  1. This is basically a memory game.
  2. Start the Game: Press any key to begin the game. Good luck!
  3. Watch the Pattern: A color will be highlighted. Memorize the sequence.
  4. Repeat the Pattern: When it's your turn, click the colors in the exact order they appeared.
  5. Add On: Each turn, one more color is added to the sequence. Keep remembering the growing pattern!
  6. Mistake? Don't worry! Restart by pressing any key again and give it another shot.
